# Noir Luxe - Premium Artisan Chocolate Landing Page

This project is a high-fidelity landing page for "Noir Luxe", a fictional premium chocolate brand.

## Features
- **Luxury Aesthetic**: Deep brown, gold, and cream color palette.
- **Responsiveness**: Fully responsive design for mobile, tablet, and desktop.
- **Animations**: Subtle fade-in animations on scroll.
- **Interactivity**: Smooth scrolling, mobile navigation menu.

## How to View
Simply open `index.html` in your web browser.

## Technologies Used
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Montserrat)
